Allow to confirm an order in the portal only if specific terms are accepted.

**Table of contents**

.. contents::
   :local:

Configuration
=============

Fill the `Terms to accept in portal` field in sale order form.

Usage
=====

In the portal view of the sale order, the signature popup shows the terms defined in the sale order.

Accepting the terms enables the Accept button.
