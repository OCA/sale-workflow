Allows storing Product Attribute values in a custom sales order line field. This can be
useful to make easier reporting and data analysys on these attributes.

**Table of contents**

.. contents::
   :local:

Configuration
=============

1. Add a custom text field to Sales Order Line
2. Create an Attributes with Variants Creation Mode set to "Never"
3. In the Attribute, set the "Store In Field" with the custem text field created

Usage
=====

On a Sales Order Line, when selecting a Product with the configured Attribute, the
corresponding value will be copied in to custom text field. This also works for
attribute custom values.
