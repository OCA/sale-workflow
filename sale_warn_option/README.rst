This module extends the functionality of Customer and Product blocking
messages to support more consistency between messages and to allow you
to set a blocking message quickly.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_

Use Cases / Context
===================

This module has been developed to maintain consistency in the messages
used when blocking customers or products in sale models.

If you need this module for those reasons, these might interest you too:

-  purchase_warn_option
-  account_warn_option
-  stock_warn_option

Usage
=====

To use this module, you need to:

-  Go to a Customer
-  Select Warn Options under *Warning on the Sales Order*
-  Go to a Product
-  Select Warn Options under *Warning when Selling this Product*
