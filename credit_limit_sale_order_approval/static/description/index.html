Credit Limit Sale Order Approval

This module was written to throw a warning when the credit limit of a customer is exceeded in a sale order, in addition to moving in this case the status to a new one named 'Waiting for approval'. If the credit limit of a customer is exceeded, only the sales manager will be able to approve the sale order.

The module also adds a new option to make optional the use of a credit limit, and shows the current state of this if it has been activated.

Installation

To install this module, you need to:

  • Check that you have installed the module sale.

Usage

To use this module, you need to:

  • Go to Sales > Sales > Customers, and open any customer.
  • Go to the Accounting tab, and check the new field Set credit limit.
  • Fill in the Credit limit field with any quantity. You will be able to see how much credit has this customer still available.
  • Save the new changes.
  • Go to Sales > Sales > Sales Orders, and create a new sale order.
  • Add new lines to the order and save it. If the total quantity is equal to or greater than the available credit of the selected customer, a message will appear at the top of the form.
  • Then you will be able to confirm the sale order, but it will pass to the state Waiting for approval.
  • When the order is waiting for approval, only a user belonging to the Sales Manager group will be able to approve the sale, clicking on the Approve Sale button.
  • Once approved, the sale order will follow the standard workflow.
