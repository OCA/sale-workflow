- Apply automatic actions:

  * Advance Payment Create

Usage
=====


To use this module, you need to:

* Go to a sale order.
* Automatic Workflow -> Automatic
* In Automatic, Checked Automated Advance Payment Create option.
* Also, Add Advance Payment Filter.
