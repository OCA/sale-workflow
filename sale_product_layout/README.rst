===================
sale_product_layout
===================

This module extends the functionality of sale_layout to automaticaly add by default
layout category on the product, and retrieve it when select a product on a sale order

Usage
=====

You must configure the default layout category on the sales tab on the product form

For further information, please visit:

* https://www.odoo.com/forum/help-1
