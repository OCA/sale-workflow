Adds a new contact's type which allows a company to mark (designate) at least one of its own contacts as the main 'ordering contact'.
This action will help in finding out who to address when dealing with sales orders.

**Table of contents**

.. contents::
   :local:

Usage
=====

#. Create/modify the desired children's contact (Contacts > Select or Create > Contacts & Addresses > Press 'Add' button) and set type to 'Ordering contact'.
#. Create a Sale Order (Sales > Create) and select the contact created/modified in 'Customer' field.
