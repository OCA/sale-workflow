Sale Product set Layout
=======================

This module was written to extend the functionality of ``sale_layout`` and
``sale_product_set`` modules and allow you to define a default *section* on
a *product set*.


Usage
=====

On product sets add default section which should be defined on sale order
lines.

For further information, please visit:

* https://www.odoo.com/forum/help-1
